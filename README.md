# FanAtlas

> **Every fan. Every story. Every place.**

FanAtlas preserves the memories of real-life encounters between sports fans around the world.

The first Atlas is **Citizen Fan Atlas**, dedicated to Manchester City supporters personally met by its founder.

Every profile represents a genuine conversation, a shared passion, and a memory worth preserving.

---

## Citizen Fan Atlas

Citizen Fan Atlas allows you to:

- 📍 Record encounters with Manchester City supporters
- 📷 Capture one photo per encounter
- 📝 Save each person's **City Story**
- 🌍 Explore all encounters on an interactive world map
- 🔎 Search and browse supporters
- 🔗 Share a public profile for each supporter

---

## Our Principles

- **KIS (Keep It Simple)**
- **Power of Concision**
- Every profile represents a **real, in-person encounter**
- The map is the heart of the experience
- Build for memories, not metrics
- Respect people's privacy
- Keep every encounter authentic
